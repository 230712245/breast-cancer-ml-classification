# Breast Cancer Classification

A machine learning project for predicting breast cancer diagnosis using multiple classification algorithms.

## Models
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Random Forest

## Features
- Data preprocessing
- Missing value handling
- Feature scaling
- Feature selection
- Pipeline integration
- Hyperparameter tuning with GridSearchCV
- Model evaluation and comparison
- Confusion matrix visualization

## Files
- `breast_cancer_classification.ipynb` → Jupyter Notebook
- `breast_cancer_classification.py` → Python script
- `best_random_forest.pkl` → saved trained Random Forest model
- `wdbc.csv` → dataset file
- `requirements.txt` → project dependencies

## Technologies
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Jupyter Notebook

## Description
This project was created to learn and compare machine learning classification models for breast cancer prediction using the Breast Cancer Wisconsin dataset.
